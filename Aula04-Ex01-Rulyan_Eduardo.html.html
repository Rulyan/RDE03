HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criadas na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo , e que podem através de hyperlinks.

TAGS

As tags definem ao navegadora marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o use de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parametros que definem algum tipo de informação extra ao navegador.
